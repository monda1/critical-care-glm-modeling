# Acceso a los Datos

Este análisis utiliza la **eICU Collaborative Research Database** (versión demo v2.0.1).

## Cómo Acceder

1. Visita: https://eicu-crd.mit.edu/

## Archivos de Datos Utilizados

- `patient.csv` - Datos demográficos de pacientes
- `vitalPeriodic.csv` - Signos vitales
- `respiratoryCharting.csv` - Parámetros respiratorios
- `apache*.csv` - Puntuaciones de severidad
